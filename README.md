 📞 Telecom Churn Prediction

## Overview
Machine learning model to predict customer churn for telecom companies.

## Features
- 85% prediction accuracy
- Identifies at-risk customers 3 weeks before churn
- $50K annual savings potential

## Project Structure
├── DataLoader.py # Load and validate data
├── Preprocessor.py # Clean and transform data
├── ModelTrainer.py # Train ML models
├── ModelEvaluator.py # Evaluate performance
├── Predictor.py # Make predictions
├── config.py # Configuration settings
└── main.py # Pipeline orchestrator
